# 量化投资（获取股票数据）

## 股票数据

### 技术面数据

追踪市场行为在价格下留下的痕迹

#### 移动平均线MA：计算一段时间内的股票平均价格

1. 简单移动平均：取一段时间内多个时间的收盘价取平均
2. 指数移动平均：增加近期股票走势的权重

移动平均线是刻画股票走势的技术指标，用于

1. 确定股票趋势
2. 交叉信号：从下方穿过移动平均线，成为黄金交叉，意味着股票可能上涨
3. 支撑与阻力线：股票接近/穿过平均线会遇到阻力，导致股票向着反方向移动

#### 相对强弱指数RSI：刻画股票市场的超买和超卖情况

计算一定时期内股票价格的平均涨幅/跌幅

1. 计算14个周期内涨幅和跌幅的平均值
2. 计算涨幅平均值和跌幅平均值之间的相对强弱比率 RS
3. 计算相对强弱比率 $RSI =100 - \frac{100}{1 + RS}$

RS越高，意味着买入股票的意愿更为强烈

#### 随机指标：价格反转的可能性

1. 计算一段时间内的最高价和最低价
2. 计算当前收盘价和最低价的差值，除以最高价和最低价的差值 $\%k= (current-min)/(max-min) * 100$
3. 计算%k平均值作为%D，计算三日%k平均

> 1. %k从下穿过%D，意味着价格上涨，应该买入
> 2. %k从上方穿过%D，意味着价格下跌，应该卖出
> 3. %k处于高位，市场超买，价格可能下降
> 4. %k处于低位，市场超卖，价格可能上涨

#### 成交量指标：刻画交易的活跃程度

某一段时间内的总成交量

1. 价格和成交量趋势相反，股票价格可能下降
2. 确认突破：价格突破支撑或阻力线，成交量大幅增加，则可能突破
3. 交易量模型：比较交易量和价格的关系作为交易信号

#### MCAD指标（Moving Average Convergence Divergence）

比较两条移动平均线的差异，判断股票价格趋势以及买卖信号

1. DIF线 短期EMA（12d）-长期EMA（26d）
2. DEA线 DIF线的平滑
3. MACD柱 DIF-DEA，显示价格的快速波动，红色表明DIF>DEA，价格上升。绿色表明DIF<DEA，价格下跌

MCAD的作用包括

1. 判断趋势（DIF和DEA交叉）
2. 买卖信号（同判断趋势）
3. 背离信号（价格和MACD指标背离，代表价格可能反向移动）

## 获取股票数据

### 获取技术面数据

1. 日线：收集每一天的开盘/最高/最低/收盘价
2. 分钟线

#### 前收盘价

需要考虑分红现金、配送股数和配股价计算前收盘价，具体而言

> 配股：按照低于市场价向原持股者按照持股比例配售一定数量新发行的股票，这么做是为了避免新发股权导致原始持有者股权被稀释，在配股缴款期内，股东有权认购一定股份
>
> 红股：利润分配时，股东将应得的利润换成股票，本质上是增加股份公司和持权股东的扩股行为，这样做能将利润用于公司发展（免费派送一些股票）

1. 除息价=收盘价-每股分红金额
2. 配股后除权价=（登记日收盘价+配股价格*每股配股数）/(1+每股配股数量) 送红股后的除权价=收盘价/(1+每股配红股数)
3. 除权除息价，先减去分红，在进行除权
